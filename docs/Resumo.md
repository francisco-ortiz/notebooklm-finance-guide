# Resumo Executivo

A Educação Financeira consiste no desenvolvimento de conhecimentos e habilidades que permitem administrar recursos financeiros de maneira consciente e planejada.

Os principais conceitos encontrados nas fontes foram:

* Planejamento Financeiro
* Receita
* Despesa
* Patrimônio
* Orçamento
* Consumo Consciente
* Reserva de Emergência
* Juros Simples
* Juros Compostos
* Investimentos

O NotebookLM foi utilizado para sintetizar essas informações, identificar conceitos recorrentes entre diferentes documentos e gerar respostas fundamentadas nas referências adicionadas ao notebook.
