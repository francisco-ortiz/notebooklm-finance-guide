# 📚 NotebookLM - Guia de Estudos em Educação Financeira

> Projeto desenvolvido para o desafio da **Digital Innovation One (DIO)** utilizando o **Google NotebookLM** para demonstrar o uso de Inteligência Artificial Generativa na organização do conhecimento, geração de resumos, engenharia de prompts e criação de materiais de estudo fundamentados em fontes confiáveis.

---

## 📖 Sobre o Projeto

Este projeto apresenta um **miniguia de estudos sobre Educação Financeira para iniciantes**, construído com o auxílio do Google NotebookLM.

Foram utilizadas fontes públicas e oficiais para demonstrar como a ferramenta pode:

* organizar informações provenientes de múltiplas fontes;
* gerar resumos fundamentados nas referências;
* criar glossários;
* comparar conteúdos;
* elaborar planos de estudo;
* produzir perguntas para revisão;
* gerar quizzes;
* criar prompts reutilizáveis para estudos futuros.

O objetivo foi demonstrar habilidades na utilização do NotebookLM, documentação técnica e Engenharia de Prompt aplicada ao aprendizado.

# 🚀 Como Reproduzir

## 1. Criar um Notebook

Acesse o Google NotebookLM e crie um novo Notebook.

---

## 2. Adicionar as Fontes

Importe entre três e cinco fontes oficiais sobre Educação Financeira.

---

## 3. Explorar o Conteúdo

Utilize prompts para:

* resumir documentos;
* comparar fontes;
* criar glossários;
* montar planos de estudo;
* gerar questões;
* criar quizzes;
* produzir mapas mentais.

---

## 4. Registrar Evidências

Salve capturas de tela demonstrando:

* prompts utilizados;
* respostas geradas;
* referências citadas pelo NotebookLM.

# 💬 Exemplos de Prompts

## Resumo

```text
Crie um resumo estruturado utilizando apenas as informações presentes nas fontes.
```

---

## Glossário

```text
Liste todos os conceitos encontrados nas fontes e organize em formato de tabela.
```

---

## Comparação

```text
Compare as fontes destacando objetivos, principais conceitos e público-alvo.
```

---

## Plano de Estudos

```text
Monte um plano de estudos de 30 minutos utilizando exclusivamente as fontes fornecidas.
```

---

## Quiz

```text
Crie um quiz com 10 perguntas de múltipla escolha utilizando apenas as informações presentes nas fontes.
```

---

## Revisão

```text
Explique os conceitos em linguagem simples para um estudante iniciante.
```
